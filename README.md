# Curry Bible Homepage

A modern, responsive homepage design for Curry Club's official website:  
https://www.currybible.co.uk

## 🔥 Overview

This project is a custom-built front-end homepage designed to:

- Showcase Curry Club reviews in a clean, modern format
- Highlight member awards (Currier of the Month & Curry C*nt)
- Promote recent reviews (Curry Chronicles)
- Integrate retro gaming (Tetris)
- Provide a strong brand identity and user experience

## 🎯 Features

- Responsive design (mobile-friendly)
- Light / Dark mode toggle (saved in localStorage)
- Modern UI with CSS Grid & Flexbox
- Clean semantic HTML structure
- Easy to scale and expand
